# 🧬 Antibiotic Resistance Gene Classification

## Overview
This project uses Machine Learning algorithms to classify DNA sequences into:

- ARG (Antibiotic Resistance Gene)
- Non-ARG

## Technologies

- Python
- NumPy
- Pandas
- Scikit-learn
- Matplotlib

## Models Used

- Logistic Regression
- Random Forest

## Dataset

Synthetic DNA sequence dataset generated for machine learning experiments.

## Results

The Random Forest model achieved promising classification performance.
